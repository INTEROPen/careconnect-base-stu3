StructureDefinition: CareConnect-Practitioner-1

The official URL for this profile is: 

https://fhir.hl7.org.uk/STU3/StructureDefinition/CareConnect-Practitioner-1

This profile defines a practitioner administration details structure that includes core localisation concepts for use in a UK context.

Identifiers

These definitions represent common data held in the Practitioner.identifier element:

  • GMC Number
  • GMP Number
  • SDS User ID

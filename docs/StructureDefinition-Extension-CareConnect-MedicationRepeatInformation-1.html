Extension: Extension-CareConnect-MedicationRepeatInformation-1

The official URL for this extension is: 

https://fhir.hl7.org.uk/STU3/StructureDefinition/Extension-CareConnect-MedicationRepeatInformation-1

Status: draft
Extension maintained by: INTEROpen

Indicates the specific repeat information of a medication item.

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"MedicationRequest"}

Extension: Extension-CareConnect-AllergyIntoleranceEnd-1

The official URL for this extension is: 

https://fhir.hl7.org.uk/STU3/StructureDefinition/Extension-CareConnect-AllergyIntoleranceEnd-1

Status: draft
Extension maintained by: HL7 UK

Supports the date and reason that the allergy was no longer valid.

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"AllergyIntolerance"}

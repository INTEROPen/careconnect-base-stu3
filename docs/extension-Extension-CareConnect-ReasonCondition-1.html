Extension: Extension-CareConnect-ReasonCondition-1

The official URL for this extension is: 

https://fhir.hl7.org.uk/STU3/StructureDefinition/Extension-CareConnect-ReasonCondition-1

Status: draft
Extension maintained by: HL7 UK - Example

The reason why a resource was added/performed/given. This may be due to a Condition, may be coded entity of some type, or may simply be present as text.

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"Observation"}

Usage info

Usage:

  • This Extension is not used in this Implementation Guide

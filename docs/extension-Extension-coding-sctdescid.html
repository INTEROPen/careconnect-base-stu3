Extension: Extension-coding-sctdescid

The official URL for this extension is: 

https://fhir.hl7.org.uk/STU3/StructureDefinition/Extension-coding-sctdescid

Status: draft
Extension maintained by: HL7 UK

This Extension resource is constrained to support SNOMED CT Description ID for the display.
